# Archive Documentation

## Overview

This archive code is basically a CMS (Content Management System) which is built using Django for the backend and React.js for the frontend.

## Status

⚠️ **Note:** Due to some configuration issues, these files are currently unusable and are uploaded as a part of development. They are provided here for reference purposes only and not to be taken into account as a part of submission for the *`Drupal FOSSEE Task`*.

## Contents

- **Backend**: Django-based REST API for managing content and events
- **Frontend**: React.js-based user interface for interacting with the CMS
- **Database**: Models and serializers for data management
- **API Integration**: Email notifications and event registration services
