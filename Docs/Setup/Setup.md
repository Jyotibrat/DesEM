# Setup Guide

This folder contains installation instructions for the different components of the *`DesEM (*`Drupal`* Custom Module)`* project. Below is the directory structure and what each folder contains:

## Directory Structure

```
Setup/
├── Setup.md                              (This file)
├── Drupal Installation/
│   ├── README Files/
│   │   └── Drupal_Installation.md       (Drupal installation instructions in Markdown)
│   └── Text Files/
│       └── Drupal_Installation.txt      (Drupal installation instructions in Text format)
├── MySQL Installation/
│   ├── README Files/
│   │   └── MySQL_Installation.md        (MySQL installation instructions in Markdown)
│   └── Text Files/
│       └── MySQL_Installation.txt       (MySQL installation instructions in Text format)
└── XAMPP Installation/
    ├── README Files/
    │   └── XAMPP_Installation.md        (XAMPP installation instructions in Markdown)
    └── Text Files/
        └── XAMPP_Installation.txt       (XAMPP installation instructions in Text format)
```

## Folder Contents

### Drupal Installation/
Contains installation instructions for setting up *`Drupal`*.
- **README Files/**: *`Drupal`* installation guide in Markdown format.
- **Text Files/**: *`Drupal`* installation guide in plain text format.

### MySQL Installation/
Contains installation instructions for setting up *`MySQL`* database server.
- **README Files/**: *`MySQL`* installation guide in Markdown format.
- **Text Files/**: *`MySQL`* installation guide in plain text format.

### XAMPP Installation/
Contains installation instructions for setting up *`XAMPP`*, which includes *`Apache Web Server`*, *`MySQL`*, *`PHP`*, and other development tools.
- **README Files/**: *`XAMPP`* installation guide in Markdown format.
- **Text Files/**: *`XAMPP`* installation guide in plain text format.
