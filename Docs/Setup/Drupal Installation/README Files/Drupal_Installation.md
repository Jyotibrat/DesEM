# Drupal Installation Guide (Windows)

1. Go to *https://www.drupal.org/project/drupal/releases/10.0.0* or [**Click Here**](https://www.drupal.org/project/drupal/releases/10.0.0).
2. Go to *`Downloads`* Section.
3. Click on *`Download zip 27.46 MB`* or [**Click Here**](https://ftp.drupal.org/files/projects/drupal-10.0.0.zip).
4. Extract the Zip File.
5. Copy the Contents of the *`drupal-10.0.0`* folder.
6. Paste it in the Project Directory inside *`C:\xampp\htdocs`*.
