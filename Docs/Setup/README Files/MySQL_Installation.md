# MySQL Installation Guide (Windows)

1. Go to https://dev.mysql.com/downloads/ or [**Click Here**](https://dev.mysql.com/downloads/).
2. At the bottom of the page click on *`MySQL Installer for Windows`* or [**Click Here**](https://dev.mysql.com/downloads/installer).
3. Select the *`Latest Version`*.
4. Select the *`Operating System`*.
5. Click on the *`Download`* option beside *`Windows (x86, 32-bit), MSI Installe (mysql-installer-web-community-8.0.45.0.msi)`* or [**Click Here**](https://dev.mysql.com/downloads/file/?id=548820).
6. In the next page, you can *`sign up`* or *`log in`* or click on *`No thanks, just start my download.`* at the bottom of the page or [**Click Here**](https://dev.mysql.com/get/Downloads/MySQLInstaller/mysql-installer-web-community-8.0.45.0.msi).
7. Go to *`File Explorer`* and navigate to *`Downloads`* folder.
8. Click on the *`mysql-installer-web-community-8.0.45.0.msi`*.
9. The *`MySQL Installer`* will start.
10. Select the *`Setup Type`* as *`Full`* and click on *`next`* or *`execute`*.
11. The Download and the Installation Process will start.
12. Once the Download is complete then the it will ask to configure the MySQL Workbench.
