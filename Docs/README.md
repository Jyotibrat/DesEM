# Docs Directory

## Overview

This is the documentation directory which contains all the setup and configuration required to use the custom module of Drupal.

## Contents

### Setup
Instructions and guides for installing and setting up the required components:
- *Drupal Installation*
- *MySQL Installation*
- *XAMPP Installation*
- *Web-App Setup* (Optional)

### Configure
Configuration guidelines for the *`Drupal custom module`*:
- *Order of Configuration*
- *Drupal Configuration*

### Assets

This directory contains all the screenshots for the testing of the custom module of Drupal.

### Module

This directory contains all the explanation of codes.

### Usage

This directory contains the guidelines how to use the `Custom Module` of `Drupal 10`.

All documentation files are provided in both Markdown and Text formats for easy reference.
